What is the Adobe Launch MCP Server?

The Adobe Launch MCP Server (also known as Adobe Experience Platform Tags MCP) is the first open registration platform that lets you generate a personal MCP Server URL and plug any AI assistant directly into your Adobe Launch workspace.

Once registered and authenticated via Adobe OAuth, your AI tool gains access to 74 Adobe Launch tools — managing tags, rules, data elements, extensions, and library publishing workflows through plain language commands.

Works with Claude, ChatGPT Desktop, Cursor, Kiro, Windsurf, and any MCP-compatible AI client.

Frequently Asked Questions

Everything you need to know

What is Adobe Launch MCP?

Adobe Launch MCP connects AI assistants to Adobe Experience Platform Launch (now called Adobe Tags / Data Collection) using the Model Context Protocol. You register once, authenticate with Adobe OAuth, and get a personal URL to use in Claude, ChatGPT, Cursor, Kiro, or Windsurf.

Is this the same as Adobe Experience Platform Tags?

Yes — Adobe Launch was officially rebranded to Adobe Experience Platform Tags and Data Collection. This MCP server supports both and gives AI access to the full suite including tags, rules, data elements, extensions, and publishing workflows.

Which AI tools are supported?

Claude (claude.ai), ChatGPT Desktop, Cursor, Kiro, and Windsurf. Any tool that supports MCP Streamable HTTP connections will work.

Are my Adobe credentials stored?

No. Your Adobe OAuth credentials are used only to validate your session and generate your MCP URL. They are never stored on our servers.

How many Adobe Launch tools can I access via AI?

74 tools — covering tag management, rule creation, data element configuration, extension management, library publishing, and property auditing.
